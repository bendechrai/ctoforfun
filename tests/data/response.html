We're about:
PHP · Mobile Development · JavaScript · MySQL · LAMP · Web Development · Software Development · Computer programming · PHP Application Development · Web Design · PHP Framework · web application development · PHP CMS

The Melbourne PHP User Group is a group of enthusiastic local programmers who get together every month and discuss Open Source technologies with a focus on PHP and associated topics. PHP is an extremely popular general-purpose scripting language that is highly suited for web development. We strive to promote PHP and Open Source in the IT industry, both locally and internationally.

Our meetings are educational and cater to all skill levels. Anyone is welcome to join us.

As a group, we aim to educate and provide a forum for professionals to get together and exchange ideas and knowledge. We have a strong leaning towards Open Source technologies.

Because of our strong focus on Open Source, we highly recommend participating in other local user groups who do the same as us, such as LUV , Melbourne Perl Mongers and OzZope to name a few. These groups all meet regularly and always have an excellent caliber of speakers.
